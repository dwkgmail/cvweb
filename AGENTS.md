# AGENTS.md

## 1. 项目定位

本项目是“丁文凯｜测试作品集网站”，目标是用于求职展示。

网站核心目的：

- 展示测试能力
- 展示电商业务测试经验
- 展示接口自动化、性能压测、SQL 校验能力
- 展示 AI 辅助测试实践能力
- 让 HR、面试官、技术负责人快速理解我的能力优势

本项目不是复杂后台系统，第一版只做静态作品集网站。

不要引入登录、权限、数据库、复杂后台管理系统。

---

默认使用：

- Vue 3

## 10. 代表性 Bug 案例

展示 3 个 Bug 卡片：

### 积分抵扣拆单退款少退

现象：按件退款后积分少退  
定位：金额整除截断导致退款积分不完整

### 支付成功订单状态未更新

现象：支付回调成功但订单未流转  
定位：回调幂等处理异常

### 趋势图日期未补零

现象：无交易日断档  
期望：按日期补 0 连续展示

## 14. 验收标准

完成后必须满足：

1. npm install 能正常安装依赖
2. npm run dev 能启动
3. 首页视觉完整
4. 页面内容与测试作品集定位一致
5. 桌面端显示正常
6. 移动端显示不崩
7. npm run build 能成功
8. dist 目录可用于 Nginx 部署
